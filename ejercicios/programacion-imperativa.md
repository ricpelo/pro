#. Encontrar el valor de la variable `valor` después de la ejecución de las
   siguientes sentencias:

   #.
      ```python
      valor = 4.0 * 5
      ```

   #.
      ```python
      x = 3.0
      y = 2.0
      valor = x ** y - y
      ```

   #.
      ```python
      valor = 5
      x = 3
      valor = valor * x
      ```
